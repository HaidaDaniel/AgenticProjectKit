# Workflows

## New project workflow

1. Initialize the kit.
2. Choose a mode.
3. Generate the initial docs and tasks.
4. Work task by task.
5. Export instructions for the preferred agent tool.

## Existing project workflow

1. Run adoption or audit.
2. Read the project structure.
3. Generate the initial docs and agent instructions.
4. Create cleanup tasks.
5. Continue with implementation tasks one by one.

## Agent workflow

1. Read the task file.
2. Read the listed context files.
3. Edit only allowed files.
4. Run verification commands.
5. Update progress and task status when complete.
