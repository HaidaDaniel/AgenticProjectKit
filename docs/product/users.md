# Users

The primary users are developers who use AI coding agents in real projects.

## Target groups

- solo developers shipping MVPs;
- small product teams;
- maintainers of brownfield projects;
- developers adopting local LLM workflows;
- teams that want consistent agent instructions across tools.

## Tools they may use

- Codex;
- Cursor;
- Claude Code;
- OpenCode;
- GitHub Copilot;
- Gemini CLI;
- local LLMs through llama.cpp, Ollama, or LM Studio.
